# FF.inc – Precision 3D Prints

A dark, modern static website for FF.inc, powered by YAML product data.

## Structure

- `index.html` — main layout
- `styles.css` — dark theme + layout
- `load-products.js` — loads YAML and injects product cards
- `data/products.yml` — product catalog
- `vercel.json` — Vercel static config
- `images/` — place your product images here

## Local Development

```bash
npm install
npm run dev
